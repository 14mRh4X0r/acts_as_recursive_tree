# ActsAsRecursiveTree

Use the power of recursive SQL statements in your Rails application.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'acts_as_recursive_tree'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install acts_as_recursive_tree
